Keychron K6 Shortcuts

This rule adds shortcuts to Keychron K6 to follow established muscle memory for window switching from a 100% layout

It also adds some extra mac specific shortcuts on less frequently used keys: Home, Page up, Page down

  • Esc → Escape
  • Shift + Esc → ` (grave)
  • Control + Esc → ~ (tilda)
  • Command + Esc → (Cycle through the current application's windows)
  • Home → Show Desktop
  • Page Up → Spotlight
  • Page Down → Mission Control
