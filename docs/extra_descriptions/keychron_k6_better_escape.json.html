Keychron K6 Better Escape Key

This rule modifies the Keychron K6 escape key to follow established muscle memory for window switching and use of ~.

The esc key will behave as follows with modifier keys:

  • Unmodified = escape
  • Shift = ~ (tilde)
  • Command = Command + ` (Cycle forward through the current application's windows)
  • Command + Shift = Command + ~ (Cycle backward through the current application's windows)
