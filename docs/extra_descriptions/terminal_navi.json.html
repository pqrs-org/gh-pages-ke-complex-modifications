Keyboard shortcuts for Terminal apps

Makes standard macOS keyboard shortcuts work in terminal apps.

Included key bindings

Shortcut Action Notes
⎋ (Esc) Abort/Cancel
⌘ Z Undo
⌘ ⇧ Z Redo [Zsh]

Zsh is the only shell that supports Redo, but unfortunately, it's not bound by default. To enable this shortcut, you need to add the following to your ~/.zshrc file:

bindkey "^[_" redo
⌥ ← Move cursor one word backward
⌥ → Move cursor one word forward
⌘ ← Move cursor to beginning of line
⌘ → Move cursor to end of line
⌥ ↑ Scroll one page up
⌥ ↓ Scroll one page down
⌘ ↑ Move cursor to beginning of buffer
⌘ ↓ Move cursor to end of buffer
⌥ ⌫ Delete one word backward
⌥ ⌦ Delete one word forward
⌘ ⌫ Delete to beginning of line

In Zsh, the command backward-kill-line does not have a keybinding by default and this shortcut will delete the whole line instead. To get it to work correctly, you need to add the following to your ~/.zshrc file:.

bindkey "^U" backward-kill-line
⌘ ⌦ Delete to end of line

Additional notes

This mod supports English and Finnish keyboards. If you want to use a different keyboard layout, please download the source code and modify terminal_navi.json.erb.
