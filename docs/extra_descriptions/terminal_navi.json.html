This mod supports English and Finnish keyboards. If you want to use a different keyboard layout, please download the source code and modify terminal_navi.json.erb.

⎋ Abort/Cancel
⌘Z Undo
⌘⇧Z Redo [Zsh]

Zsh is the only shell that supports Redo, but unfortunately, it's not bound by default. To use the shortcut listed here, you need to add the following to your ~/.zshrc file:

bindkey "^[_" redo

⌥↑ Scroll one page up [Emacs, Less]
⌥↓ Scroll one page down [Emacs, Less]
⌘↑ Move cursor to beginning of buffer or file
⌘↓ Move cursor to end of buffer or file
⌥⌫ Delete one word backward
⌥⌦ Delete one word forward
⌘⌫ Delete to beginning of line

In Zsh, the command backward-kill-line does not have a keybinding by default and the shortcut for "Delete to beginning of line" will delete the whole line instead. To get it to work correctly, you need to add the following to your ~/.zshrc file:.

bindkey "^U" backward-kill-line
