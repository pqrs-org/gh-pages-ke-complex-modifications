Dual keys

This mod relieves stress from your pinkies by moving several hard-to-reach keys to more ergonomic positions. It accomplishes this by making several keys do double duty as both modifier keys and input keys, depending on whether your press them alone or hold them down.

Combines well with the Ergoemacs mod.

Notes

This mod is made for using a US layout with an ISO (not ANSI) MacBook keyboard. If you want to use a different keyboard layout or otherwise remap the keys, please download the source code and modify dual_keys.json.rb.
